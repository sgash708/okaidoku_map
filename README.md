# 概要
日常の買い物を安く済ませたい人向けのアプリケーションを実装しています。
このアプリケーションでは、営業しているスーパーの特売情報を取得して表示します。

# できること
* マップ機能による検索
    * 現在値検索
    * キーワード検索(商品/店舗)
    * ユーザ登録/更新/削除/確認
    * ユーザのログイン・アウト
    * お気に入り店舗の登録
    * 店舗一覧表示/詳細表示
    * 店舗の表示(営業中/休業)
